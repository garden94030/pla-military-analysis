# 伊朗-中國STS石油轉運網絡與制裁規避

## 事件摘要

2026年4月3日，OSINT分析師Charlie B（@supbrow）發布基於哥白尼Sentinel-2衛星影像的分析報告，揭露馬來西亞東部港外錨地（EOPL）發生的大規模伊朗-中國石油STS轉運活動。衛星偵測到6起STS轉運，其中3起確認為伊朗→中國石油轉運；所有作業油輪均關閉AIS；12艘涉案船舶中6艘列入UANI幽靈艦隊黑名單，2艘懸掛伊朗旗。

## 關鍵數據

| 指標 | 數值 |
|------|------|
| 偵測STS轉運 | 6起（3起確認伊朗→中國） |
| AIS關閉率 | 80/180艘（44%） |
| 幽靈艦隊黑名單 | 6/12艘 |
| 伊朗旗油輪 | 2艘 |
| 自伊朗戰爭起抵達EOPL | ~19艘伊朗旗油輪 |
| 離港 | ~24艘 |

## 戰略意涵

伊朗石油收入持續支撐IRGC軍事能力和代理人網絡，在伊朗-美國直接軍事衝突背景下具有重要戰略意義。暗艦隊向假旗和無旗籍船舶轉型，顯示制裁規避技術持續升級。
