# 公安部第三研究所Telegram監控能力

## 事件摘要

2026年4月4日，「中国人事观察」（@cnpoliwatch）在X平台揭露，中國公安部第三研究所正在積極展示一套針對Telegram平台的大規模監控工具。工具宣稱已累積超過300億條Telegram消息，監控7,000萬個帳戶，涵蓋39萬個公共頻道與群組。監控重點為使用中國手機號碼（實名制）註冊的Telegram帳戶，並宣傳具備偵測VPN使用行為的配套工具能力。開發動因之一明確指向2019年香港「反修例運動」中抗議者的Telegram使用。

## 關鍵數據

| 指標 | 數值 |
|------|------|
| 收集消息總量 | 超過300億條 |
| 監控帳戶數量 | 7,000萬個 |
| 監控頻道/群組 | 39萬個 |
| 主要目標 | 中國手機號碼實名制帳戶 |

## 威脅評估

此工具的開發和展示，標誌著中共數字威權主義技術能力的重要躍升。核心威脅在於：以手機號碼實名制為突破口的身份識別能力、社交網絡圖譜重建能力，以及對海外異見社群（香港、維吾爾族、藏族）的穿透滲透能力。台灣公民社會和情報相關人員面臨間接暴露風險。
