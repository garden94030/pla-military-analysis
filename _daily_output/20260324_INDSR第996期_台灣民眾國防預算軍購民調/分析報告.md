# INDSR《國防安全即時評析》第996期
## 台灣民眾對國防預算與軍購特別預算之態度分析

**來源：** 國防安全研究院（INDSR）
**作者：** 方琮嬿助理研究員（中共政軍與作戰概念研究所）
**發布日期：** 2026年3月23日
**關鍵字：** 國防預算、民意調查、軍購特別預算、不對稱戰力
**焦點類別：** 國防安全民意調查
**調查執行：** 國立政治大學選舉研究中心（2026年3月12-16日電話調查，n=1,169）

---

## 一、調查背景

行政院2025年11月27日通過《強化防衛韌性及不對稱戰力計畫採購特別條例》草案，規劃自2026年起八年投入1.25兆元，採軍購與國造並行，重點籌獲：
- 精準火砲
- 無人載具
- 防空飛彈
- 反裝甲飛彈

本次民調評估台灣民眾對此政策的認知與支持程度。

---

## 二、調查主要發現

### 2.1 「投資國防就是投資和平」認同度

| 群體 | 認同比例 |
|------|---------|
| **全體** | **約59%** |
| 男性 | 64% |
| 女性 | ~54% |
| 40-49歲 | 65%（最高） |
| 民進黨支持者 | 87% |
| 國民黨支持者 | **約33%**（不認同55%，高於認同） |
| 民眾黨支持者 | <52% |

### 2.2 「軍購特別預算」支持度

| 態度 | 比例 |
|------|------|
| **支持（非常/有點）** | **62%** |
| 不支持 | 22% |
| 無反應 | 17% |

**重要發現：** 對「抽象理念」（投資國防=和平，59%認同）的支持低於對「具體政策」（特別預算，62%支持），顯示台灣民眾在面對具體的不對稱戰力強化方案時，支持意願更為明確。

### 2.3 關鍵族群分析

**年輕族群（18-29歲）：**
- 支持比例最低（59%）
- 無反應比例最高（25%）
- 顯示年輕人對軍購議題仍欠缺明確立場

**政黨因素：**
- 民進黨支持者：支持度92%
- 國民黨支持者：支持度33%（黨派差距最大）
- 中立/無特定傾向：約50%支持、20%以下不支持

---

## 三、分析評估

### 3.1 民意基礎相對穩健

逾六成民眾支持特別預算，在台灣政治分極化的背景下屬於相對高的跨黨派共識，顯示**威脅認知已在相當程度上凝聚社會共識**，超越單純的黨派立場。

### 3.2 黨派分歧仍是最大變數

民進黨（92%）與國民黨（33%）之間的近60個百分點落差，是台灣國防預算政治的最大風險點。若政黨輪替，特別預算的持續推進面臨不確定性。

### 3.3 年輕世代的政策溝通缺口

18-29歲有25%「無反應」，顯示這一世代缺乏充分的國防認知教育。鑒於年輕世代是未來的兵役基礎，提升其對不對稱作戰戰略的理解是當務之急。

### 3.4 對中共心理戰的抵禦意涵

超過六成民眾支持強化防衛能力，意味著中共以「和平談判可解決爭端」為核心的統一戰線宣傳，在台灣社會的穿透效果有限。這一數字對中共的「認知域作戰」評估具有直接參考價值。

---

## 四、政策建議

1. **持續深化跨黨派對話**：將軍購政策的戰略必要性去政治化，強調台灣防衛能力是兩岸關係穩定的基礎
2. **針對年輕族群的國防教育**：整合學校課程與社群媒體，提升18-29歲對不對稱作戰的認知
3. **軍購透明化**：定期向社會說明特別預算的採購進度與效益，維持民意基礎
