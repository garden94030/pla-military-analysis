## 2. 核心分析

### 關鍵發現

根據日本防衛研究所森木愛太研究員的報告，中共已系統性地將大量退役的殲-6戰鬥機改裝為無人自殺攻擊載具。這項發現揭示了中共採用「數量壓制」戰略，利用成本相對低廉的老舊戰機改裝，形成大規模無人機群攻擊能力。

### 對台灣防衛的潛在影響

此一威脅對台灣防空體系構成嚴峻挑戰。殲-6改裝的自殺無人機具備以下特徵：首先，數量龐大且成本低廉，能形成飽和攻擊；其次，可能在衝突初期大量消耗台灣珍貴的防空飛彈庫存；第三，迫使台灣防空系統面臨「昂貴飛彈攔截廉價目標」的不對稱困境。

### 與中共軍事發展的關聯

這項軍事改裝反映中共軍事現代化的務實策略：善用既有軍備庫存、發展不對稱作戰能力，以及整合無人作戰系統到整體戰略中。此舉符合中共「多域作戰」概念，透過低成本手段達成戰略目標，同時展現其在無人機技術應用上的快速發展。

## 3. 政策建議清單

### 針對國防部門
- **多層次防空體系建構**：發展成本效益更佳的近程防空武器，如機砲、雷射武器系統
- **電子作戰能力強化**：投資電子干擾與網路作戰能力，癱瘓無人機導航與通訊
- **彈藥庫存管理**：重新評估防空飛彈配置策略，避免被低價值目標消耗高價值武器
- **預警系統升級**：強化雷達網路，提早發現大規模無人機群攻擊

### 針對民防規劃
- **民眾防護教育**：加強空襲警報系統與避難演練
- **關鍵基礎設施防護**：強化機場、港口、電力設施的點防禦能力
- **應急復原機制**：建立快速修復與替代系統的民防動員體制

### 針對國際協調
- **情報分享機制**：與美日等盟友建立無人機威脅情報交換平台
- **技術合作發展**：參與多國反無人機技術研發計畫
- **區域防空整合**：推動第一島鏈防空預警網路整合
